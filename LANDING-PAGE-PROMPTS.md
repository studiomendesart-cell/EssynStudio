## NOTAS PARA O FIGMA MAKE

- Usar Instrument Serif para headlines (disponivel no Google Fonts como "Fraunces" ou "Playfair Display" como alternativa)
- Inter para body text
- Icones: Lucide React (Aperture, Wallet, CalendarDays, Images, UserRoundSearch, etc.)
- Cores primarias: #0A0A0B (fundo), #FFFFFF (texto), #9C8B7A (gold accent), #007AFF (blue CTA)
- Mockups do app devem usar screenshots reais ou representacoes fieis do UI
- Animacoes: scroll-reveal suave, parallax sutil, hover scale nos cards
- Mobile-first: todas as secoes devem funcionar em mobile (stack vertical, carousel)
